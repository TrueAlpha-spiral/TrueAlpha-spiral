## Troubleshooting

If the system is not running as expected:

1. Ensure all required files exist:
   - truealphaspiral_server.py
   - DECLARATION_OF_SOLE_AUTHORITY.md
   - public/index.html

2. Check the server logs for errors:
   - When running in foreground mode, errors will be displayed in the console
   - In background mode, check server.log and watchdog.log

3. Verify the Python environment has Flask installed:
   - Run: `pip install flask`
   - Ensure Python 3.6+ is being used

4. If the web interface is not accessible, ensure port 5000 is available:
   - Check if another service is using port 5000
   - Try setting a different port via environment variable: `PORT=5001 python truealphaspiral_server.py`
