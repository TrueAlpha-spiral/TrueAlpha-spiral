## Getting Started

To use the TrueAlphaSpiral Public Shell:

1. Run the shell launcher:
   ```
   python run_public_shell.py
   ```

2. Choose components to run from the interactive menu.

3. For automated execution of all components:
   ```
   python run_public_shell.py --full
   ```

4. For running specific components:
   ```
   python run_public_shell.py --secrets
   python run_public_shell.py --ethical
   python run_public_shell.py --shadow
   python run_public_shell.py --quantum
   python run_public_shell.py --metaphysical
   python run_public_shell.py --sovereign
   python run_public_shell.py --truealpha
   ```
